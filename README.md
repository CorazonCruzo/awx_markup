# AWX Test Task — Страница вывода средств

Мобильная страница вывода средств, выполненная в рамках тестового задания для позиции Middle React Developer.

## Demo

[https://awx-markup-9nh8.vercel.app](https://awx-markup-9nh8.vercel.app)

## Технологии

- React 18
- TypeScript
- Material UI 5
- Feature-Sliced Design (FSD)

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка

```bash
npm run build
```
